# 项目简介

## 核心功能
- 基于YOLOv8的人脸识别与打卡功能
- 支持多种录入方式：单张图片录入、批量文件夹录入、摄像头实时录入
- 提供本地摄像头打卡功能
- 活体检测参数可自定义设置
- 打卡状态和信息实时显示
- 支持多种检测模型选择
- 可自定义置信度和IOU阈值

## 适用场景
- 企事业单位员工考勤管理
- 学校学生签到系统
- 社区门禁识别
- 会议参会人员验证
- 需要进行人脸识别打卡的各类场所

## 技术特点
- 采用Python 3.9开发，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整的UI界面，操作简便
- 包含7000张图片的标注数据集
- 训练好的模型可直接使用，无需重新训练
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
